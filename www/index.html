Список функций

http://localhost:8090/api/game/join/

Войти в игру; параметры: Name; ответ 1 - успешно, 0 - неуспешно

http://localhost:8090/api/game/leave/

Покинуть игру; параметры: Name; ответ 1 - успешно, 0 - неуспешно

http://localhost:8090/api/game/ready/

Зашло ли два игрока; параметры: нет; ответ 1 - оба игрока готовы, 0 - второго игрока нет

http://localhost:8090/api/game/isWin/

Кто победил?; параметры: нет; ответ 0 - никто не победил, Строка имени победителя

http://localhost:8090/api/game/add/

Сделать ход; параметры: Имя игрока,координата x, координата y;ответ 1 - успешно, 0 - неуспешно

http://localhost:8090/api/game/cells/

Возвращает все заполненные клетки в формате JSON; параметры:нет;ответ [{"login":"","x":"","y":""}]

http://localhost:8090/api/game/next/

Чей следующий ход?; параметры:Имя игрока;ответ 1 - ход введенного игрока, 0 - ход другого игрока

http://localhost:8090/api/game/restart/

Обновляет поле?; параметры:нет;ответ 1 - успешно обновлено, 0 - неуспешно
